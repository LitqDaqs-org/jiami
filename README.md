# Jiami (甲醚) - A Personally Designed Encrypted Text Website

## What's Jiami (甲醚)

Jiami (甲醚) is a lightweight, browser-based encrypted text website designed for technical exploration and proof-of-concept demonstrations. The name "甲醚" reflects its core purpose: providing a simple platform for encrypting and decrypting text content within a web browser environment.

## About Jiami (甲醚)

Jiami is built with HTML, CSS, and JavaScript, utilizing client-side encryption to process user data. The project focuses on:

- **Local Processing**: All encryption and decryption operations occur directly in your browser
- **Simple Interface**: A user-friendly design for easy text encryption/decryption
- **Technical Exploration**: Demonstrates modern web cryptography capabilities
- **Educational Purpose**: Serves as a learning resource for developers interested in encryption techniques

The project was collaboratively created as a personal exploration of encrypted web technologies, combining technical innovation with practical implementation.

### Project Statistics
- **Pages**: HTML (12%)
- **Styling**: CSS (4.1%)
- **Functions**: JavaScript (83.9%)
